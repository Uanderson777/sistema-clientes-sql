# 🧑‍💻 Sistema de Gerenciamento de Clientes

Sistema desktop desenvolvido em Python com interface gráfica utilizando Tkinter e banco de dados SQLite.

---

## 📌 Sobre o Projeto

Este projeto tem como objetivo demonstrar habilidades em desenvolvimento de aplicações desktop, manipulação de banco de dados e criação de interfaces gráficas.

O sistema permite realizar operações completas de CRUD (Criar, Ler, Atualizar e Deletar) de clientes.

---

## 🚀 Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Atualização de dados
- Exclusão de registros
- Interface gráfica intuitiva
- Banco de dados local (SQLite)

---

## 🛠 Tecnologias Utilizadas

- Python 3
- Tkinter
- SQLite3
